# wireframes.md — Jira Feature Report Tool

All wireframes are structural ASCII layouts. Styling and colours are not specified here.

```
+------------------------------------------------------------------+
| 🗂  Jira Feature Report                                           |
+----------------+-------------------------------------------------+
| SIDEBAR        | MAIN CONTENT                                    |
|                |                                                 |
| Open Statuses  | Enter a Jira Feature ticket ID to generate      |
| (multi-select) | the Excel report.                               |
| [ To Do      ] |                                                 |

| [ In Review  ] | +-------------------------------------------+  |
| [ + Add more ] | | MTC-1234                                  |  |
|                | +-------------------------------------------+  |
|                |                                                 |
|                | [ Generate Report ]                             |
|                |                                                 |
|                | ------------------------------------------------|
|                | ℹ️  Configure open statuses in the sidebar      |
|                |    before generating.                           |
+----------------+-------------------------------------------------+
```
